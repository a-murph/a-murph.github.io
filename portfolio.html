MyMovieList

MongoDB, Express, React, Node

Modeled after MyAnimeList.com, MyMovieList is a full-stack MERN app that allows users to search for movies using the OMDB API and add them to their personal lists. Users can list each movie as "Completed", "Plan-to-Watch", or "Dropped" and rate them from 1 - 10, and easily share their list with friends or online.


Technologies used: MongoDB, Mongoose, Express, React, React Router, Node, Axios, OMDB API, Bootstrap, Materialize.

Polygon Scraper

MongoDB, Express, Node

This site scrapes the front page of Polygon.com and displays the headlines, thumbnails, and bylines of each new article. Users can then comment on any given article, or go to the archives to view all scraped articles.


Technologies used: MongoDB, Mongoose, Express, Node, Cheerio, Handlebars

React Memory Game

React
